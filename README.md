### AI GitHub Contributor Automation Agent

https://hub.docker.com/repository/docker/jahnavik186/ai-github-contributor-automation-agent/general

An AI-powered automation system that analyzes repositories, identifies issues, and assists with generating pull requests. The project demonstrates how AI agents can automate developer workflows and improve open-source contribution efficiency.

---

### AI Observability Platform

https://hub.docker.com/repository/docker/jahnavik186/ai-observability-platform

A self-hosted observability platform for AI and LLM applications that tracks latency, token usage, estimated cost, failures, and quality signals. The project demonstrates how teams can monitor AI systems with a reusable stack built on FastAPI, PostgreSQL, Redis, Grafana, and Docker.

---

### ICU Early Risk Predictor

https://hub.docker.com/repository/docker/jahnavik186/icu-early-risk-predictor

A containerized predictive AI system that estimates ICU admission risk using clinical patient features. The system exposes a FastAPI-based prediction service for real-time clinical decision support and demonstrates scalable deployment of healthcare machine learning models.

---

These projects showcase how machine learning systems can be **packaged, deployed, and scaled using containerized infrastructure for real-world applications.**
